# Observability Platform

Enterprise observability platform with metrics, logs, traces, and custom dashboards.

## Quick Start

```bash
# Start backend
cd observability-backend
mvn spring-boot:run

# Open frontend
open observability-frontend/pages/index.html
```

## Features

- **Overview** - System health dashboard
- **Dashboards** - Custom dashboard builder
- **Metrics** - Metrics exploration
- **Logs** - Real-time log viewer
- **Traces** - Distributed tracing
- **Services** - Service health monitoring
- **Alerts** - Alert management

## Configuration

Edit `config/constants.js` to change API URL and settings.
